Hubungkan Google Sheets

LemburApp menyimpan data ke Google Sheets via SheetDB.io (gratis). Ikuti langkah di bawah, lalu paste URL API-nya.

Cara setup (5 menit):

1. Buka sheets.google.com → buat spreadsheet baru
2. Buat 2 sheet tab dengan nama persis:
   • accounts
   • overtimes
3. Di sheet accounts, baris 1 isi header:
   id | name | empno | username | password | role | initials
4. Di sheet overtimes, baris 1 isi header:
   id | accId | date | start | end | type | reason | status | empSig | adminSig | createdAt
5. Buka sheetdb.io → Sign Up gratis → Create API
6. Paste link Google Sheets tadi → copy URL API yang muncul
7. Paste URL di atas ↑
⚠️ Pastikan nama sheet tab ditulis huruf kecil persis: accounts dan overtimes
